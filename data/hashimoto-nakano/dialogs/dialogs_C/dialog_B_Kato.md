# 後輩との関係
- 問題点:
  - 後輩が指示を聞かない
  - 後輩との関係がキャリアに影響しないか心配
- 影響:
  - 仕事の効率が下がる可能性
  - チーム全体の連携が乱れる
- 指示が無視される場面:
  - 緊急時
  - 忙しい時

# 後輩との関係
- 問題点:
  - 後輩が指示を理解してくれない
  - 質問をせず自己判断で動く
- 影響:
  - 薬の投与方法を間違える
  - 患者対応でのミスが多い
- 解決策:
  - 質問しやすい環境作り
  - 定期的に話す時間を設ける
    - 実施方法: シフトの合間や休憩時間に週1回短時間の面談

# 専門看護師資格の取得
- 目標: 専門看護師資格の取得
- 問題点:
  - 後輩との関係が不安
  - 勉強時間の確保が難しい
- 現状のスケジュール:
  - 平日: フルタイムで勤務
  - 休日: 映画鑑賞や小説執筆
- 工夫:
  - 映画鑑賞や執筆の時間を減らして勉強時間を確保
- 課題:
  - 試験が近づくと勉強時間が足りず焦りを感じる
