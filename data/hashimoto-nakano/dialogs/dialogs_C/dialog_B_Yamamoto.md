# 時間管理の工夫
- 使用しているツール:
  - スケジュール帳
  - デジタルカレンダー
- 方法:
  - 時間管理を徹底し、優先順位を明確化
  - 仕事の合間や休日を活用して効率的に作業を進める

# 資格取得と学習
- 目標:
  - 専門知識を深めるために資格取得を目指している
- 勉強方法:
  - 勉強時間を確保する計画を立て、仕事の合間や休日を活用
- 勉強と仕事のバランス:
  - 休憩時間を有効活用し、集中力を維持する
  - リフレッシュ方法:
    - 短時間の瞑想
    - ストレッチ
    - 深呼吸
    - 軽い読書
    - 音楽を聴く

# スキル向上の取り組み
- 参加する研修:
  - フィジカルアセスメント・アドバンス研修
- 目的:
  - 患者さんの状態をより正確に把握するスキルを習得する
