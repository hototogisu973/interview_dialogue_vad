# 忙しさとやりがい
- 忙しさ: 最近特に忙しく、疲れが溜まっている
- やりがい: 患者さんに感謝されるとやりがいを感じる

# 職務内容
- 職場: がん患者が多い外科病棟
- 満足点: 仕事にやりがいを感じている
- 不満点: 疲れが溜まりやすい

# キャリアの課題
- 課題: スキルアップの機会が少ない

# スキルアップの希望
- 伸ばしたいスキル:
  - がん看護の専門知識
  - 緩和ケアの技術
- 希望する学びの方法:
  - 研修やセミナーへの参加
  - 専門書を読む

# 必要なサポート
- サポートの必要性: スキルアップの機会を増やすための環境
