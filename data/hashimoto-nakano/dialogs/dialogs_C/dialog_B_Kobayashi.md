# 忙しさと体調管理
- 忙しさ: 特に夜勤が多く、疲れが溜まっている
- 体調管理: 疲れを感じているが、体調管理が課題

# 転職の検討
- 転職を考えた理由:
  - シフトの長さと精神的な負担が大きく、限界を感じている
  - 夜勤の頻度とストレスが理由
- 異動希望:
  - 異動ではなく、看護業界を離れることを検討
- 看護業界を離れる理由:
  - 看護業界のストレスが大きい
  - 他の職種に興味がある

# 興味のある職種
- 興味のある職種:
  - ヨガインストラクター
  - 手芸関連の仕事
- 理想の職場や環境:
  - ヨガスタジオや手芸店など、リラックスできる環境

# 転職準備状況
- 準備状況:
  - 情報収集を始めたばかりで、具体的な準備はまだ

# 必要なサポート
- 異動後に必要なサポート:
  - メンタルケア
    - 具体的な方法:
      - 定期的なカウンセリング
      - リラクゼーションの時間

# 理想の転職環境
- 理想の職場:
  - ストレスの少ない環境
  - 定時で働ける職場
  - 趣味との両立が可能な環境
